## نبذة عن النظام
هو نظام إداري متكامل مصمم لأتمتة عمليات الموارد البشرية وإدارة الموظفين داخل المنشآت. يهدف النظام إلى تنظيم الهيكل الإداري وتحويل العمليات الورقية إلى تجربة رقمية سريعة وآمنة، مع ضمان أعلى معايير الأمان وعزل البيانات بناءً على الأقسام (Department Isolation).

من خلال هذا النظام، يمكنك:
* **إدارة الموظفين بذكاء:** إضافة الموظفين وتوليد أرقام وظيفية تلقائية تعتمد على أقسامهم (مثل IT, HR).
* **التحكم بالصلاحيات (Roles):** توزيع الأدوار بين (مدير، مشرف، موظف)، حيث يشرف كل رئيس قسم على موظفيه فقط.
* **أتمتة الطلبات الإدارية:** تقديم ومتابعة طلبات الإجازات وتذاكر الدعم الفني مع رفع المرفقات والمستندات.
* **تقييم الأداء الاحترافي:** بناء نماذج تقييم تعتمد على أوزان ومعايير محددة، وحساب النتائج والتقديرات آلياً.
* **التفاعل اللحظي:** استلام إشعارات فورية (Real-time) عند صدور القرارات دون الحاجة لتحديث الصفحة.
* **حماية البيانات:** إجراء نسخ احتياطي دوري وشامل لكامل بيانات النظام والمرفقات.

## الأدوات التقنية (Tech Stack)
**Backend:** Laravel 13 | **Real-time:** Laravel Reverb (WebSockets)

**Frontend:** Tailwind CSS, Alpine.js | **Database:** MySQL (Indexed)

**Storage:** Laravel Storage (Symlink) | **Queueing:** Database Queue Driver

**Security:** Spatie Permissions | **Reporting:** mPDF (Tajawal Arabic Font)

**Backup:** Spatie Backup (Automated)

## نظام الأدوار والصلاحيات (Access Control)

### 1. مدير النظام (Super Admin)
* **التحكم المطلق:** الوصول الكامل لكافة الأقسام، الإعدادات، ومسيرات الرواتب الحساسة.
* **الأمان:** تنفيذ عمليات النسخ الاحتياطي (Backup) وإدارة صلاحيات المستخدمين.

### 2. مشرف القسم (Supervisor)
* **عزل الأقسام:** لا يرى إلا الموظفين والتذاكر وطلبات الإجازة التابعة لقسمه فقط.
* **التقييم الذكي:** إدارة تقييمات الأداء لموظفي فريقه المباشر حصراً.

### 3. الموظف (End User)
* **الخدمة الذاتية:** إدارة الملف الشخصي، تقديم طلبات الإجازة، وفتح تذاكر الدعم.
* **الشفافية:** استعراض وتحميل تقارير أدائه الرسمية بصيغة PDF.

## المميزات التفصيلية لكل صفحة

### لوحة التحكم (Dashboard)
* إحصائيات لحظية وتوزيع الموظفين عبر الأقسام العشرة المعتمدة.
* تفعيل نظام الـ Caching لضمان سرعة التحميل تحت ضغط البيانات العالي.

### إدارة الموظفين
* **الرقم الوظيفي الذكي:** توليد تلقائي يعكس هوية القسم (مثل IT-0001).
* **إدارة المرفقات:** رفع الصور الشخصية مع نظام تنظيف تلقائي للملفات القديمة.

### نظام التذاكر والإجازات
* دعم المرفقات المتعددة (صور، فيديو، مستندات) وتتبع المسار الزمني للحالات.
* نظام ردود لحظي يضمن سرعة التواصل بين الموظف ومشرفه.

### تقييم الأداء الاحترافي
* حساب آلي للنسب المئوية والتقديرات اللفظية الملونة بناءً على مؤشرات الأداء (KPIs).
* تصدير التقارير بتنسيق PDF رسمي متوافق مع اللغة العربية.

## تعليمات التشغيل (Terminal Commands)
شغل الأوامر التالية في 4 نوافذ Terminal منفصلة:

**السيرفر الرئيسي:**
```bash
php artisan serve
```

**الإشعارات اللحظية:**
```bash
php artisan reverb:start
```

**معالج المهام (Queue):**
```bash
php artisan queue:work --tries=3
```

**تجميع الواجهة:**
```bash
npm run dev
```

## الصيانة والدورية

**النسخ الاحتياطي اليدوي:**
```bash
php artisan backup:run
```

**ربط الملفات العامة:**
```bash
php artisan storage:link
```
